Abstract

 In recent years, infection with new coronavirus has spread all over the world. It is said that the number of tests such as PCR for this infectious disease has not caught up with the number of infected people. So, we need a simpler and lower cost inspection method. The new test kit we propose is “Ordinary people can easily do it at home” using HCR. By amplifying with HCR, it is used at home without the need to use a dedicated machine like PCR. In addition, this mechanism is caused by a specific reaction with the spike protein on the surface of the new coronavirus, so there is no need to extract RNA in the new coronavirus and even ordinary people without knowledge can do it. The result is expressed by the color change of the test solution using the aggregation effect of gold nanoparticles, so it is visualized and anyone can easily understand it.(近年新型コロナウイルスの感染が世界中に拡大しています。そして、この感染症に対するPCRなどの検査の検査数は感染者数に追いついていないと言えます。また、検査に必要な機材は専門的かつ高価です。そこで、より簡易的で低コストな検査方法が必要になります。私たちが提案する新たな検査キットは、HCR法を用いた「家庭で一般の人でも簡単に」検査できるキットです。HCRを用いて二本鎖DNAを、イニシエーターと2種類のヘアピンDNAの伸長反応によって増幅を行うことで、PCRのように専用の機械を使う必要がなく家庭で新型コロナウイルスの有無の検出を行えます。またこの反応は、新型コロナウイルス表面のスパイク質との特異的な反応により起こるためRNAの抽出を行う必要がなく、家庭に機材がない状態で行えます。そしてその結果は、金ナノ粒子の凝集効果を用いた検査溶液の色の変化で表現するため、簡単に理解できます。)

この検査キットが実現すれば、感染の有無が分かるようになり感染者との接触を減らす事ができる。また、感染していないことが分かれば行動を過度に制限する必要が減り、動きやすくなる。その結果効果的な抑え込みができ、感染者を減らしていくことができるとようになると期待できます。他にも、この検査キットは自宅で検査できるため、検査のために医療機関を訪れるなどといった外出を避けられ、さらなる感染拡大を防ぐことができるようになると考えられます。
